# Node Discord MusicBox

## Getting Start
### Source
nodejs + typescript

### package script
```bash
# clean build source
npm run clean

# build source
npm run build

# start server
npm run serve

# stop server
npm run stop

# copy key file
npm run copy
```

### library
- discord.js
- nodeopus
- ytdl
- ffmpeg
